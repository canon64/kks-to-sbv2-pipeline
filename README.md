# KKS to SBV2 Pipeline

KoikatsuSunshine のボイスを抽出し、Style-Bert-VITS2 でファインチューニングするための一体型GUIツールです。

## 機能概要

| タブ | 内容 |
|------|------|
| 抽出 | KKS AssetBundle から WAV ファイルを抽出 |
| DB構築 | 抽出した WAV から SQLite データベースを構築 |
| ブラウズ | DB を閲覧・絞り込み・CSV エクスポート |
| SBV2トレーニング | Style-Bert-VITS2 でファインチューニング |

## 前提条件

- **Style-Bert-VITS2** が別途インストール済みであること
  - SBV2 ディレクトリを「SBV2ルート」欄に指定する
- KoikatsuSunshine 本体（AssetBundle が必要）
- Windows 10/11 (x64)
- Python は不要（setup.bat が自動でローカル配置します）

## セットアップ

### 初回のみ

```
setup.bat
```

`python/` フォルダに Python 3.11 をダウンロードして展開します（システムには一切インストールしません）。

### 起動

```
run_kks_all_in_one_gui.bat
```

## 使い方

### 1. 抽出タブ

1. **KKSディレクトリ** に KoikatsuSunshine のインストールフォルダを指定
2. **抽出開始** を押すと `wave/` フォルダに WAV が出力される

### 2. DB構築タブ

1. WAVディレクトリ・DB保存先を確認（抽出タブから自動補完）
2. **DB構築開始** を押す

### 3. ブラウズタブ

1. DB ファイルを開く
2. キャラ・感情・レベルで絞り込み
3. **エクスポート** で選択した WAV と CSV を出力フォルダにコピー

エクスポート完了後、SBV2トレーニングタブの WAV/CSV パスが自動補完されます。

### 4. SBV2トレーニングタブ

| 設定項目 | 説明 |
|----------|------|
| SBV2ルート | Style-Bert-VITS2 のインストールディレクトリ |
| WAVソース | エクスポート済み WAV フォルダ |
| CSVソース | エクスポート済み CSV（セリフ一覧） |
| モデル名 | 任意のモデル識別名 |
| エポック数 | トレーニングのエポック数（デフォルト: 10） |
| 評価間隔 | チェックポイント保存ステップ数（デフォルト: 1000） |
| バッチサイズ | GPU メモリに合わせて調整（デフォルト: 2） |
| GPU | 使用するGPU（nvidia-smi / rocm-smi で自動検出） |

**エポック数の目安**: 819サンプル・バッチサイズ2の場合、1エポック≒410ステップ。5000〜10000ステップを目標にするなら 12〜25 エポック程度。

チェックポイントは `model_assets/{モデル名}/` に保存されます。再開時はエポック数を現在値より大きく設定してください。

```
└── trainer/                 # SBV2トレーニング機能
```
